ShareOn is a collection of simple lightweight, stylish and ethical share buttons for WordPress.

== Description ==

ShareOn is a lightweight share buttons originally created by <a href="https://github.com/NickKaramoff">Nick Karamoff</a>.

To use ShareOn, add the ShareOn widget to the your sidebar and select your desired sharing networks. The script will automatically share the current page url.
